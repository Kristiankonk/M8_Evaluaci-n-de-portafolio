Sobre mí

Soy Cristian Ramírez, desarrollador Full Stack en formación, con experiencia en el desarrollo de aplicaciones web utilizando tecnologías como Django, MySQL, HTML, CSS, JavaScript y Bootstrap. Me interesa especialmente el diseño de sistemas que mejoren la eficiencia, la organización y la calidad de los procesos.

A través de este portafolio presento proyectos que responden a problemáticas reales, poniendo en práctica buenas prácticas de desarrollo, control de versiones con Git y despliegue en entornos productivos.

Proyectos Destacados

Tienda de Coleccionables

Aplicación web desarrollada con Django y MySQL para gestionar un catálogo de productos coleccionables. Incluye carrito de compras, control de stock, flujo de checkout y panel administrativo.

  • Problema: gestión y venta organizada de productos coleccionables.
  • Tecnologías: Django, MySQL, Bootstrap, HTML, CSS.
  • Funcionalidades: autenticación, carrito, pedidos, CRUD de productos y categorías.
Ver código en GitHub Ver demo

Sistema de Reservas

Sistema web de reservas que permite gestionar servicios, horarios y reservas de usuarios, con control de disponibilidad y panel de administración.

  • Problema: organización de reservas y horarios para un servicio.
  • Tecnologías: Django (o stack a elección), Bootstrap, base de datos relacional.
  • Funcionalidades: registro de usuarios, reservas, gestión de servicios y agenda.
